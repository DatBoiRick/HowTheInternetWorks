About the Assignment

The objective of this assignment is to get familiarized with how the internet works. This particular assignment is accomplished individually. A website must be created (similar to the one in the previous assignment) and explain the following topics: Domain names and ICANN, IP Addresses,Packets and Routing, DNS, HTTP and HTTPs protocols, W3C and HTML and CSS.
